# Collaborative Code Editor

A real-time collaborative code editor that allows multiple users to write and edit code together. It provides a VS Code-like editing experience with instant synchronization across users.

---

## Features

- Real-time multi-user collaboration  
- Monaco Editor integration  
- Live synchronization using Yjs  
- WebSocket communication with Socket.IO  
- Room-based session sharing  
- Responsive UI  

---

## Tech Stack

### Frontend
- React.js  
- Monaco Editor  
- Yjs  
- Socket.IO Client  

### Backend
- Node.js  
- Express.js  
- Socket.IO
